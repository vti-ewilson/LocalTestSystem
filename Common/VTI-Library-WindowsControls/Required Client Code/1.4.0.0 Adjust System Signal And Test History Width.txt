Settings Described
- Client Side Properties Added through Settings.settings, See scope and type below
	- Adjust test history font from client
		+ Properties.Settings.Default.TestHistoryFont
		+ Properties.Settings.Default.TestHistoryLabelWidth
		+ Properties.Settings.Default.TestHistoryLabelHeight
	- Adjust test history count when not docked from client
		+ Properties.Settings.Default.TestHistoryColumns
		+ Properties.Settings.Default.TestHistoryRows
	- Adjust  signals font from client 
		+ Properties.Settings.Default.SystemSignalCaptionFont
		+ Properties.Settings.Default.SystemSignalValueFont
	- Adjust the system signal panel width from client
		+ Properties.Settings.Default.SystemSignalPanelWidth
	- Ability to adjust the valve panel font and width from client
